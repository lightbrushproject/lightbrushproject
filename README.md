## Focus

> Lightbrush builds tools and systems for artists, venues, performers, and creative operators working with projected light, responsive visuals, spatial media, and real-time environments.

| Area | What it supports |
| --- | --- |
| Projection systems | Mapping, show control, content playback, and installation prep. |
| Live visuals | Resolume, FFGL, GLSL, generative scenes, and operator-friendly effects. |
| 3D motion | Animated identity systems, WebGL studies, procedural loops, and branded visual worlds. |
| Creative automation | Pipelines that turn media, prompts, and assets into usable production outputs. |
